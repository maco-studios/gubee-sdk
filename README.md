# About

This SDK is designed to facilitate the integration between your application and the Gubee API, providing a simplified interface to interact with the various endpoints available within the Gubee platform.

## Requirements

- PHP 7.4 or higher
- Composer for dependency management

## Installation

Use Composer to install the SDK into your project:

```bash
composer require gubee/sdk
```
